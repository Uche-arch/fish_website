Creating a simple JavaScript carousel involves combining HTML, CSS, and JavaScript. Below is a simple example of how you can create a basic carousel with previous and next buttons to navigate between images.

Here’s the structure for the carousel.

<html lang="en">

css

3. JavaScript

Here’s the JavaScript to make the carousel functional.

javascript
// script.js

let index = 0;

function moveSlide(step) {
  const slides = document.querySelectorAll(".carousel-image");
  index += step;

  if (index >= slides.length) {
    index = 0;
  }
  if (index < 0) {
    index = slides.length - 1;
  }

  const carousel = document.querySelector(".carousel");
  carousel.style.transform = `translateX(-${index * 100}%)`;
}

Explanation:

  1. HTML:

    • The .carousel-container holds the entire carousel and its controls.
    • The .carousel contains the images.
    • The buttons prev and next are used to navigate between images.

  2. CSS:

    • The .carousel-container has overflow: hidden; to ensure only one image is visible at a time.
    • The .carousel has display: flex; to make the images appear in a row, and the transition property helps in smooth sliding between images.
    • The buttons are styled to appear on the left and right sides of the carousel.

  3. JavaScript:

    • The moveSlide() function adjusts the index variable, which determines the current slide.
    • It uses the transform property with translateX to slide the carousel left or right based on the button clicked.

How It Works:

  • Clicking the "previous" button (<) decreases the index, showing the previous image.
  • Clicking the "next" button (>) increases the index, showing the next image.
  • The transform property is used to move the entire carousel by updating the translateX value.
